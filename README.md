É um sistema que automatiza via reconhecimento de imagem;

O Código deve ser inscrito no arquivo "app.py" na função "aplication";

As funções podem ser encontradas na pasta módulos;

As imagens que serão usadas devem ser guardadas na pasta "assets/imagens".

O sistema deve ser iniciado pelo arquivo "interface.py"

Para criação de executavel recomenda-se usar pyinstaller.
